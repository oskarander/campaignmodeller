# AX Revenue Lab - Full Suite Comparator

A sophisticated revenue strategy dashboard for comparing baseline pricing models with challenger simulations. This tool provides deep insights into transaction shares, revenue variance, and PAX capacity trends.

## Features

- **Dynamic Simulation:** Real-time calculation of revenue and profit based on transaction shares.
- **Visual Feedback:** Interactive sliders and high-contrast indicators for model validation.
- **Strategic Metrics:** Tracks Incremental Profit, Flex Revenue, and PAX Delta.
- **Mobile Responsive:** Modern, premium design that works across devices.

## How to Host on Git

To host this project on GitHub Pages:

1. Create a new repository on GitHub.
2. Initialize this folder as a git repo:
   ```bash
   git init
   git add .
   git commit -m "Initial commit of AX Revenue Lab"
   ```
3. Add your remote:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   ```
4. Push your code:
   ```bash
   git push -u origin main
   ```
5. In your GitHub repository settings, go to **Pages** and select the `main` branch to host the site.
